An Automated Platform to Mitigate the Impact of Road accidents

What is SmartCop ?

It is an automated platform to mitigate the impact of road accidents. This proposed system contains four modules.

  • 01 Predict road accidents

    The road accident prediction module is implemented to predict accidents’ severity, reason, frequency, and locations. For prediction purpose the Random forest classifier and KNN algorithms are used.

  • 02 Scheduling traffic police officers

    The “SmartCop – Traffic police officers scheduling” module is implemented with two main functionalities respectively, recommending appropriate police officers based on their experience analysis using K Nearest Neighbors algorithm and Cosine Similarity, and Scheduling the recommended officers using optimization. To make the scheduling more effective, the recommendation and scheduling process was carried out according to the results of the road accidents prediction module.

  • 03 Enhance road accidents prevention awareness

    The road accidents prevention awareness module is implemented to enhance the public awarenss on road accident prevention. Upon a publicly conducted survey, a hangman game has been selected for enhancing public awareness. Questions of the game are classified as easy/difficult based on the prediction retrived from the pre-trained Support Vector Machine binary classification model. Furthermore, next levels of this game built dynamically in real-time based on the previous level user performance (easy/moderate/difficult) tracked by the Logistic Regression multi class classification model.

  • 04 Enhance post road accidents response awareness

    The “SmartCop – Safe Life Quiz” game is an android mobile application which is developed to improve post road accident response knowledge among people. The application is implemented using Flutter and machine learning. The main feature of this game is questions that will change dynamically by analyzing the player’s performance. The questions and time need for question to answer are predicted using Random forest classifier algorithm.
